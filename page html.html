تاریخچه HTML

مقدمه

HTML (HyperText Markup Language) زبان نشانه‌گذاری استاندارد برای ایجاد صفحات وب است. این زبان در اوایل دهه 1990 میلادی توسط تیم برنرز لی (Tim Berners-Lee) ابداع شد.

تاریخچه

اولین نسخه HTML در سال 1991 منتشر شد و فقط شامل 18 عنصر بود. این عناصر برای ساختاردهی به متن‌ها، ایجاد لینک‌ها، و درج تصاویر مورد استفاده قرار می‌گرفتند. HTML به سرعت تکامل یافت و در طول زمان بهبودهای بسیاری در آن ایجاد شد.

HTML 2.0 در سال 1995 معرفی شد و شامل استانداردهای جدیدی برای طراحی وب بود. سپس HTML 3.2 در سال 1997 و HTML 4.01 در سال 1999 معرفی شدند. هر کدام از این نسخه‌ها ویژگی‌ها و قابلیت‌های جدیدی را به زبان HTML اضافه کردند.

HTML5

HTML5 جدیدترین نسخه HTML است که در سال 2014 به صورت رسمی منتشر شد. HTML5 قابلیت‌های جدیدی مانند پشتیبانی از ویدئوها، انیمیشن‌ها، و برنامه‌های تحت وب را به این زبان اضافه کرد. همچنین، HTML5 تلاش کرد تا کد نویسی صفحات وب را ساده‌تر و استانداردهای وب را یکپارچه‌تر کند.

نتیجه‌گیری

HTML همچنان در حال توسعه و بهبود است. امروزه این زبان به عنوان اصلی‌ترین ابزار برای ساخت و توسعه وب سایت‌ها مورد استفاده قرار می‌گیرد. با یادگیری و استفاده از HTML، شما می‌توانید صفحات وب حرفه‌ای و کارآمدی ایجاد کنید.
